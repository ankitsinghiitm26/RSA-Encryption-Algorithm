Title: RSA Encryption Algorithm
This project demonstrates the basic principles of asymmetric cryptography, key generation, encryption, and decryption. RSA (Rivest–Shamir–Adleman) is a famous public-key encryption algorithm used in real-world security systems.
Unlike Caesar cipher or substitution cipher:
•	Public key → used to encrypt messages 
•	Private key → used to decrypt messages 
Simple Flow:
Original Message
       ↓
Encrypt using Public Key
       ↓
Encrypted Data
       ↓
Decrypt using Private Key
       ↓
Original Message
